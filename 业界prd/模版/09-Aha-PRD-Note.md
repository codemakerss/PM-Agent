# Aha! PRD Note 模板

来源：https://www.aha.io/roadmapping/guide/templates/create/prd

## 适用场景

适合产品经理在产品管理系统中记录产品、版本、Epic 和 Feature 需求。

## 模板结构

- Product Requirements：产品级目标、愿景、用户、核心能力
- Release Requirements：版本目标、时间、范围、依赖
- Epic Requirements：Epic 背景、目标、范围
- Feature Requirements：功能说明、验收标准、状态、负责人

## BA 使用要点

- 适合从高层目标逐级拆解到功能。
- BA-Agent 可按产品、版本、Epic、Feature 四层输出。
- 需求评审时可直接转入路线图管理。
